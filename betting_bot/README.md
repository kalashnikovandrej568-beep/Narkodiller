# Telegram Betting Bot 🎲

Полноценный Telegram-бот с системой ставок и управлением балансом пользователей.

## Быстрый запуск (одна команда в PowerShell)

```powershell
cd "C:\Users\Admin\VSC программы\betting_bot"; pip install -r requirements.txt; python main.py
```

> **Примечание:** перед запуском создайте файл `.env` в папке `betting_bot` с содержимым:
> ```
> BOT_TOKEN=ваш_токен_от_botfather
> ```

## Характеристики

✅ Регистрация пользователей с начальным балансом  
✅ Система ставок на события с реальными участниками  
✅ **Система силы участников (1-10) с автоматическим расчётом коэффициентов**  
✅ **Автоматический случайный выбор победителя на основе силы**  
✅ Таблица лидеров  
✅ История ставок  
✅ Админ-панель для управления событиями  
✅ Дневной бонус (10-50 монет один раз в 24 часа)  
✅ Система трансфера денег между пользователями (админ может отправлять по User ID)  
✅ Защита админ-панели паролем с запоминанием  
✅ SQLite база данных  
✅ Асинхронная обработка команд

## Установка

### 1. Клонировать репозиторий

```bash
git clone <URL репозитория>
cd betting_bot
```

### 2. Установить зависимости

```bash
pip install -r requirements.txt
```

### 3. Настроить .env файл

Отредактируйте файл `.env` и добавьте ваш токен Telegram:

```
BOT_TOKEN=ваш_токен_от_botfather
```

### 4. Запустить бота

```bash
python main.py
```

## Получение токена

1. Откройте Telegram и найдите [@BotFather](https://t.me/botfather)
2. Отправьте `/newbot`
3. Следуйте инструкциям
4. Скопируйте полученный токен в `.env`

## Админ-панель

Для доступа нажмите кнопку **👨‍💼 Админ** и введите пароль: `32415`

### Функции админа:
- ➕ **Создать событие** - создать новое событие со списком участников и указать силу каждого (1-10)
- 📋 **Список событий** - просмотреть все события с коэффициентами
- 🏁 **Закрыть событие** - завершить событие (победитель выбирается автоматически на основе силы)
- 💸 **Перевести деньги** - отправить деньги пользователю по его User ID

### Система силы и коэффициентов:

При создании события админ задаёт **силу** каждого участника от 1 до 10 (неограниченное кол-во участников).  
Система автоматически рассчитывает шансы и коэффициенты выплат.

**Пример (2 участника):**
- Боец А: сила 1, Боец Б: сила 10  
  → Шанс А = 9.1%, коэф = x11.00  
  → Шанс Б = 90.9%, коэф = x1.10  

**Пример (3+ участника):**
- Боец А: сила 7, Боец Б: сила 2, Боец В: сила 1  
  → Шанс А = 70%, Шанс Б = 20%, Шанс В = 10%

При закрытии события **админ НЕ выбирает победителя** — победитель выбирается автоматически рандомом с учётом силы.

### Дополнительные фичи:

- 📊 **Визуальные полоски шансов** — `████████░░` вместо сухих цифр
- 🔥 **Серия побед/поражений** — отображается в профиле, если 2+ подряд
- ⚠️ **Защита от двойных ставок** — нельзя ставить дважды на одно событие
- 💰 **Статистика ставок** — видно кол-во ставок и общий пул на каждом событии
- 📋 **Полный список событий** — активные + последние закрытые

### Старые администратор-команды:
- `/admin` - админ-панель
- `/admin_create_event <название> <коэффициент>` - создать событие
- `/admin_list_events` - список событий
- `/admin_close_event <ID события> <победитель>` - закрыть событие
- `/admin_get_stats <user_id>` - статистика пользователя
- `/admin_set_balance <user_id> <сумма>` - установить баланс

## Конфигурация

Отредактируйте `config.py` для изменения параметров:

- `BOT_TOKEN` - токен Telegram
- `ADMIN_IDS` - список ID админстраторов
- `INITIAL_BALANCE` - начальный баланс (по умолчанию 100)
- `MIN_BET` - минимальная ставка (по умолчанию 10)
- `DATABASE_PATH` - путь к БД

## Пример использования

1. Пользователь пишет `/start` и получает 100 монет
2. Админ создает событие через кнопку, вводит участников и их силу
3. Пользователь нажимает "🎲 Ставки", выбирает событие и на кого ставить
4. Админ нажимает "🏁 Закрыть событие" — победитель выбирается **автоматически**
5. Выигрыши начисляются по коэффициенту: ставка × (сумма_сил / сила_победителя)
